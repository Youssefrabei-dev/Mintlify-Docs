Sawa is a social activity planning platform. These docs are the backend source of truth for developers working on the Express API, database schema, Better Auth integration, generated OpenAPI reference, and mobile contract pipeline.

The current focus is English-first and backend-first: setup, architecture, file ownership, API contracts, and everyday backend workflows.
